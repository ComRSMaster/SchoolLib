# SchoolLib

#### Это проект электронной библиотеки ФТШ на Python

## Автор:
* Кирилл Муравьёв

Backend написан на Flask.<br>
Frontend написан без фреймворков, с библиотекой CSS - Bootstrap.<br>
Для базы данных используется SQLite
